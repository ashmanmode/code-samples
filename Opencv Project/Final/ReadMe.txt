-> Input file format
   The input file should contain the address of the image from the present directory and one line should contain address of only one image

-> Interactive Tool
   Interactive tool is a console based application where user will be prompted to draw some shapes. User will be needed to enter the name 
   for desired shape and then draw on the image canvas. This tool will record all the necessory user inputs.

1. Name will start with C and will be of format "CName".

1. All the function names should be of format "FunctionForDoingSomething".

// NAMING style for variables in the classes

1. All the variable names should use underscore as a separator. e.g variable_name
